# **App Name**: Ngecet

## Core Features:

- Direct Messaging: Enables users to create direct, one-on-one chats with other users.
- Group Messaging: Allows users to create and join group chats for multiple participants.
- Local Storage: Messages are stored locally in the user's browser using local storage.
- Chat List: A list of active chats (direct and group) with the most recent at the top for easy access.
- User Profile: Users can set a profile name and optionally an avatar image. Avatars could be generated using AI.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to evoke a sense of calm and connection. Chosen to create a simple and elegant feel.
- Background color: Light gray (#F5F5F5) for a clean and unobtrusive backdrop.
- Accent color: Muted purple (#B39DDB) used sparingly for highlights and interactive elements, providing a subtle contrast.
- Clean and modern sans-serif fonts for a sleek and readable interface.
- Simple and minimalist icons for common actions, such as sending messages, adding users, and muting notifications.
- Clean and spacious layout with clear divisions between sections to improve readability and user experience.
- Subtle animations and transitions to enhance the user experience and provide feedback for interactions.
